# ガントチャート機能設計

## 機能概要

タスクの期間・進捗をテキストガントチャートとしてSlackに投稿・更新する機能。

---

## データ設計

新規SQLite（`gantt.db`）に `tasks` テーブルを作成する。

| カラム名 | 型 | 説明 |
|---|---|---|
| id | INTEGER | 主キー |
| name | TEXT | タスク名 |
| start_date | TEXT | 開始日（MM-DD） |
| end_date | TEXT | 終了日（MM-DD） |
| status | TEXT | 未着手 / 進行中 / 完了 |
| note | TEXT | ボトルネックメモ |
| posted_at | TEXT | ボトルネック投稿日時（MM-DD HH:MM） |

---

## 表示フォーマット

```
タスクA  ████░░░░░░  3/24 - 3/27
タスクB  ░░░████░░░  3/26 - 3/29
タスクC  ░░░░░░████  3/28 - 4/01
```

- `█` : タスク期間
- `░` : 期間外
- 表示幅は全タスクの最小開始日〜最大終了日に合わせる

---

## 操作フロー

1. ホットキーでタスク入力フォームを表示
2. タスク名・開始日・終了日・ステータスを入力
3. DBに保存
4. SlackのチャンネルへASCIIガントチャートを投稿（既存メッセージがあれば更新）

---

## 技術仕様

### スクリプト側
- DB：新規SQLite（`gantt.db`）
- ガントチャート生成：Pythonで文字列を組み立て
- 日付計算・表示幅の管理
- n8n Webhookへデータ送信（`n8n.py`）

### Dify側
- 殴り書きの整形・タスク名の正規化
- ボトルネックの要約
- Slack投稿（Difyノード使用）
